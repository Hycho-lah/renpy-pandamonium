Bantuan Dasar

Untuk maju dalam permainan, klik-kiri atau tekan tombol spasi atau enter. Pada saat berada di menu, klik-kiri untuk memilih, atau gunakan tombol panah untuk memilih dan gunakan enter untuk mengaktifkan pilihan.

Menu Permainan

Ketika bermain game, kilik-Kanan atau tekan tombol escape untuk masuk ke menu permainan. Menu permainan terdiri dari beberapa pilihan berikut:

Kembali
Kembali Ke Permainan.
Save Game
Mengijinkan kamu untuk menyimpan permainan dengan mengklik save slot
Load Game
Mengijinkan kamu untuk meload permainan dengan menglik save slot. Mengklik "Load.C" mengakses slot save otomatis.
Opsi
Mengubah pengaturan permainan (opsi / konfigurasi):
Display
Beralih antara mode Layar Penuh dan Jendela
Transisi
Mengatur transisi layar antara layar permainan
Kecepatan Text
Mengendalikan kecepatan text di tampilkan. Jika slider lebih jauh ke kanan, text akan lebih cepat di tampilkan, jika sudah penuh ke kanan text akan di tampikan secara instant
Joystick
Memungkinkan kamu untuk mengendalikan permainan menggunakan Joystick.
Loncati
Pilih untuk melompati text yang telah terlihat (Pesan Terlihat), atau lompati Semua Pesan.
Mulai Loncati
Kembali ke permainan, selagi melompati.
Setelah Pilihan
Mengendalikan, apakah melompati berhenti setelah mencapai menu/pilihan
Waktu Auto Forward
Mengendalikan maju otomatis. Semakin jauh slider ke kiri, semakin lambat waktu yang di butuhkan sebelum game maju. Jila sudah full ke kanan itu berarti text tidak akan pernah secara otomatis maju ke text berikut nya.
Musik, Souara, dan Volume Dialog/Suara
Mengendalikan volume Music, Efek Suara, dan Channel Dialog/Voice, secara terpisah. Semakin jauh slider ke kanan, semakin keras suara nya.
Menu Utama
Kembali ke menu utama, mengakhiri game yang sedang berlangsung.
Bantuan
Menampilkan Layar Bantuan Ini.
Keluar
Keluar permainan; game akan di tutup dan di akhiri.

Binding Mouse Dan Keyboard

Klik-kiri, Enter
Maju di dalam permainan, mengaktifkan menu pilihan, tombol, dan slider.
Spasi
Maju di dalam permainan, tapi tidak mengaktifkan pilihan.
Tombol Panah
Navigasi di antara menu pilihan, tombol, dan slider.
Ctrl
Menyebabkan melompati aktif ketika tombol ctrl di tahan.
Tab
Menyala/matikan lompati, menyebabkan melompati aktif sampai tombol di tekan kembali.
Mousewheel-Up, PageUp
Menyebabkan kembali aktif. Kembali menyebabkan game mundur ke text sebelumnya, dan bahkan mengijinkan untuk mengubah pilihan yang telah di pilih.
Mousewheel-Down, PageDown
Menyebabkan tidak kembali aktif, membatalkan aksi kembali yang telah di lakukan.
Klik-kanan, Escape
Masuk ke menu permainan.Ketika sedang di menu permainan berubah menjadi kembali ke permainan.
Klik-tengah, H
Menyembunyikan jendela text, dan hud transparan lainnya.
F
Menyala/Matikan mode Layar Penuh
S
Mengambil tangkapan layar, menyimpannya di file bernama screenshotxxxx.png, dimana xxxx adalah nomor seri.
Alt-M, Command-H
Menyembunyikan (iconifies).
Alt-F4, Command-Q
Keluar permainan.
Delete
Ketika slot save terpilih menghapus save di slot tersebut.
v
Menyala/matikan mode self-voicing, yang akan membacakan text ke user mengunakan speech synthesizer bawaan os. Untuk info lebih lanjut tolong baca dokumentasi self-voicing
Shift+C
Menyala/matikan mode clipboard-voicing, yang mengkopi text ke papanklip jadi dapat dibaca oleh screen reader.

Binding Kontroller

Game ini seharusnya secara otomatis mendeteksi dan menggunakan kontroller game yang didukung oleh SDL2. Kontroller lain dapat di konfigurasikan dengan tool pihak ke tiga seperti SDL2 Gampad Tool

Sedikit komputer mungkin akan mengalami masalah ketika mendeteksi kontroller. Jika itu terjadi, tekan shift ketika game dimulai, dan matikan dukungan kontroller.

Berikut binding yang di gunakan di kontroller:

Right Trigger, A (Tombol Bawah)
Maju di dalam permainan, mengaktifkan menu pilihan, tombol, dan slider.
Guide, Start
Masuk ke menu permainan.Ketika sedang di menu permainan berubah menjadi kembali ke permainan.
Directional Pad, Stik Analog
Navigasi antara pilihan menu, tombol, dan slider.
Left Trigger, Left Shoulder, Back
Menyebabkan kembali aktif. Kembali menyebabkan game mundur ke text sebelumnya, dan bahkan mengijinkan untuk mengubah pilihan yang telah di pilih.
Right Shoulder
Menyebabkan tidak kembali aktif, membatalkan aksi kembali yang telah di lakukan.
Y (Tombol Atas)
Menyembunyikan jendela text, dan hud transparan lainnya.
